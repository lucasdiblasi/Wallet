# Wallet
Carteira de investimentos em flutter

## Funcionalidades:

* Gráfico de investimento em carteira.
* Saldos na tela com função de esconder valores.
* Lista de dados de posição consolidada.
* Gráfico de rentabilidade de carteira.

## Linguagem 

* Dart.

## Framework

 * Flutter.
